# Portfolio

Welcome to my portfolio! I am an illustrator with experience working on various projects for a wide range of clients. My
work spans across character design, product illustrations, unique locations, and seamless patterns. Below, you will find
examples of my work, organized into different categories.

_Some images contain watermarks as per the client’s request for displaying them publicly._
